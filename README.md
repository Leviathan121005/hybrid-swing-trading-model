# Hybrid Swing Trading Model

A single-stock swing trading model that learns trading policies from historical price data. The model operates under a next-open decision framework, where trading actions are determined based on information available up to the previous day close and applied at the following opening price. During training, however, the model learns from price sequences that include both past and future time steps. The model consists of three components:

## 1. Naive Bayes Trend Indicators Classification

The model uses three technical indicators to represent short-term trend conditions:

- Relative Strength Index (RSI-14)
- Commodity Channel Index (CCI-20)
- Williams %R (14)

These indicators are used as input to a Naive Bayes classifier, which classifies them into profitable trade actions (Buy, Sell, Hold) generated by evaluating the current price position over a fixed look-ahead window (`peek` days). Once trained, the classifier outputs a trade action based on the current indicator values.

---

## 2. K-means Price State Clustering

Market conditions are represented using five normalized price-action features:

- Open change relative to previous close
- High change relative to same day open
- Low change relative to same day open
- Close change relative to same day open
- Volume change relative to previous day volume

K-means clustering groups these observations into *k* discrete clusters. Each cluster represents a market state, allowing continuous price behavior to be represented as a finite state space.

---

## 3. SARSA Reinforcement Learning Agent

The SARSA agent learns the trading policy through interaction with historical data.

- **State:** Price state cluster from K-means  
- **Actions:** Buy, Sell, Hold  
- **Policy:** Epsilon-greedy exploration  
- **Constraints:**
  - Action space restricted to classifier-suggested action or Hold
  - Only one position can be held at a time
  - Stop-loss enforced at 10%

The agent learns a state-action value function (Q-table) that maximizes cumulative returns over time.

---

## Summary

This architecture combines supervised learning, unsupervised learning, and reinforcement learning into a unified trading system that learns both market structure and optimal trading behavior.
